# Happy Birthday App 🎂

Este é o meu primeiro projeto prático no curso de **Desenvolvimento Android Nativo** do Google. 

O objetivo principal foi aprender os fundamentos do **Jetpack Compose** e como estruturar uma interface moderna no Android.

## 🚀 O que eu aprendi neste módulo:
* **Composables:** Criação de funções de UI reutilizáveis.
* **Layouts:** Como organizar elementos usando `Column`, `Row` e `Box`.
* **Recursos:** Como adicionar e manipular imagens da pasta `res/drawable`.
* **Modificadores:** Ajuste de alinhamento, preenchimento e escala de conteúdo (`ContentScale.Crop`).

## 🛠️ Tecnologias Utilizadas
* Kotlin
* Jetpack Compose
* Android Studio
